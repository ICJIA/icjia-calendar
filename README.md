# icjia-calendar

> Agency calendar using [Vue](https://vuejs.org/) and [Strapi](https://strapi.io/).

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
